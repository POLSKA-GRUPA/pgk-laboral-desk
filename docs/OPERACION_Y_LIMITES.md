# Operacion y limites

## Como usar la herramienta

La consulta debe incluir, idealmente:

- puesto o categoria
- jornada
- modalidad
- temporada o periodo de actividad si se plantea fijo-discontinuo
- antiguedad o trienios
- prorrata o no de pagas extra

Ejemplo valido:

```text
Quiero contratar un socorrista nivel B fijo discontinuo a 40 horas semanales para la temporada de verano.
```

Ejemplo incompleto:

```text
Socorrista media jornada fijo discontinuo
```

En el segundo caso el sistema responde, pero pedira aclaraciones porque no puede cerrar categoria exacta ni periodo real de actividad.

## Como leer la salida

### 1. Lo que he entendido

Resume la lectura estructurada del caso:

- categoria
- modalidad
- jornada
- trienios
- pagas extra
- punto sensible

### 2. Conclusion del caso

Es la respuesta operativa prioritaria para direccion:

- si hay encaje provisional
- si faltan datos
- si puede emitirse pre-nomina orientativa

### 3. Pre-nomina orientativa

Sirve para obtener una cifra base de convenio del mes activo. No es una nomina definitiva.

### 4. Condiciones aplicables

Son los bloques juridico-laborales relevantes del convenio ya ordenados para consulta.

## Limites juridicos

La herramienta no debe usarse por si sola para:

- emitir nomina definitiva
- cerrar contrato internacional
- validar permisos de trabajo
- resolver cotizacion o fiscalidad transfronteriza
- sustituir el analisis del Estatuto de los Trabajadores o normativa superior

## Limites tecnicos

- solo trabaja con un convenio cargado
- no persiste expedientes
- no soporta autenticacion
- no integra base de datos
- no calcula deducciones legales

## Recomendacion de uso interno

Secuencia correcta:

1. Direccion formula la necesidad de contratacion.
2. La herramienta da lectura, condicionamiento y base economica.
3. El equipo laboral valida:
   - convenio aplicable
   - modalidad correcta
   - jornada y calendario
   - impacto de Seguridad Social e IRPF
4. Si hay componente internacional, se abre expediente separado de extranjeria y fiscalidad.
